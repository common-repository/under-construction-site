== Installation ==

1. Upload the entire `under-construction-site` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Setting menu and then open under Construction sub menu to customize this plugin.
4. Finally Enjoy it.

== Frequently Asked Questions ==

= Under construction page is not displayed? =

Check that your Under construction page mode is enabled or not, if it's enabled then enable it.  Now open your site url in a different browser where you are not logged.

= Under construction mode is enabled but i can't see the page =

If you are using any cache plugin then first deactivate it now check your site in different browser.
